选择排序教学演示

选择排序算法步骤：

  1. 从未排序的元素中找出最小(或最大)的元素
  2. 将它与未排序部分的第一个元素交换位置
  3. 将该元素视为已排序
  4. 重复以上步骤，直到所有元素都已排序
C语言实现
// 选择排序函数 void selectionSort(int arr[], int n) { int i, j, min_idx; // 遍历未排序的数组 for (i = 0; i < n-1; i++) { // 找到未排序部分中的最小元素 min_idx = i; for (j = i+1; j < n; j++) { if (arr[j] < arr[min_idx]) min_idx = j; } // 将找到的最小元素与未排序部分的第一个元素交换 if (min_idx != i) { int temp = arr[i]; arr[i] = arr[min_idx]; arr[min_idx] = temp; } } }
